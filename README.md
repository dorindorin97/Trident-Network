# Trident Network (TRI)

A community-driven Trident Network fork with modified consensus and branding.

## Specifications
- **Block Time:** 2 seconds
- **Total Supply:** 1,000,000,000 TRI
- **Chain ID:** 0x76a81b116bfaa26e

## Quick Start

### Build Locally
```bash
# Requires Docker
docker build -t trident-network:latest .
```

### Launch Local Devnet
```bash
docker-compose up
```
